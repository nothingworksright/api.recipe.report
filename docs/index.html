This is the source code repository for the Recipe.Report API Server.

Start Recipe.Report API Server in development mode:

First, prepare a development copy of the Recipe.Report database. Run the vagrant up command to create and configure a Vagrant VM running PostgreSQL. Run the yarn run flyway command to download Flyway. Run the yarn run migrate command to apply database migrations.

vagrant up
yarn run flyway
yarn run migrate

Next, prepare and build the source code. Run the yarn run prettier command to format the source code using the Prettier opinionated styling. Run the yarn run linter command to identify and report ESLint code pattern findings. Run the yarn run test command to check for dependency updates, run Jest unit tests, and create coverage reports. Run the yarn run build command to compile the TypeScript into plain JavaScript, and generate the Typedoc documentation.

yarn run prettier
yarn run linter
yarn run test
yarn run build

Finally, run the yarn run develop command to start the Recipe.Report API Server in development mode.

yarn run develop

Start Recipe.Report API Server in production mode:

 
yarn run start
